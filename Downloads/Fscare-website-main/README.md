# FS Care Website

> **Your Trusted Partner in Health and Wellness** — a modern React website for FS Care pharmacy.

## Tech Stack
- React 18
- CSS Modules (no Tailwind dependency)
- Lucide React (icons)
- Google Fonts (Playfair Display + DM Sans)

---

## 🚀 Deploy to GitHub + Vercel (Step-by-Step)

### 1. Push to GitHub

```bash
# In this folder:
git init
git add .
git commit -m "Initial commit: FS Care website"

# Create a new repo on github.com called "fscare-website", then:
git remote add origin https://github.com/YOUR_USERNAME/fscare-website.git
git branch -M main
git push -u origin main
```

### 2. Deploy on Vercel

1. Go to [vercel.com](https://vercel.com) and sign in with GitHub
2. Click **"Add New Project"**
3. Import your `fscare-website` repository
4. Vercel auto-detects Create React App — just click **Deploy**
5. Your site is live in ~60 seconds! 🎉

## 🛠 Local Development

```bash

```
